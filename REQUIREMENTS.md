# Requisitos

- **Node.js 22.5+** (o back-end usa `node:sqlite`, nativo do Node — sem
  dependências externas de banco de dados). Testado na 24.x.
- **npm** (vem junto com o Node).
- Uma conta na **Bybit** com uma API key **somente leitura** (mainnet),
  cadastrada depois pela tela de Configurações do app — veja o passo a
  passo no [README](README.md).

Só para deploy em produção (Fly.io):

- **Docker**
- **flyctl** e uma conta na Fly.io
